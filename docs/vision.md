## Как игра должна выглядить (начало)
1. меню игры
играть
настройки
выход
2. начало игры (возможно предыстория мира-персонажа)
игрок должен спавниться в основном городе (гл.замке) где его встречает король со своими подданными (идет рассказ)
надо придумать рассказ история мира персонажа и его задачи в этом мире 
3. игрок выходит из (гл.замка) на главную площать где по центру находиться фантан и есть несколько вариантов пути 1. торговцы с права от игрока 2. в лево от игрока классовые персонажи 3. люк с канализацией на главной площяди 4. вниз выход из города (сначала ворота города) потом локация у ворот лес.
4. на доске с объявлениями в центре главной площяди задание по убийству мини босса у города в лесу 
5. король отправил убить пару крыс в канализации (мб для отчистки канализации (будет решаться когда будет продумываться сюжет))
6. после канализации подходим к королю и дальше идет история (история будет продумываться дальше)
следующий квест выход из города в лес убийство каких нибудь волков(змей) (дать возможность ходить дальше)
если игрок пойдет дальше может умереть от высокоуровнивых мобов (но есть возможность посмотреть на мини босса по квесту на доске объявлений)

# Архитектура Pet_Games_2D
data(ресурсы игры(звуки, картинки, музыка))/
docs(документация)/architecture.md (архетиктура всего проекта где какой файл и что к чему превязано), roadmap.md (дорожная карта проекта что должно быть в игре какие задачи стоят), vision.md (текстовая визуализация игры что будет в игре и как это должно выглядить (ланшафт, нпс, логика, диалоги, задумка игры)),
src(файлы игры)/main.py (Основаная логика игры)
venv/ - изолированная среда с зависимостями
readme.md - основной файл обложка гита (лицо игры)
requirements.txt - требования для запуска проекта

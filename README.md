# Pet_Games_2D
Ведеться разработка игры (Игра в первую очередь делается для обучения python)
Игра будет представлять из себя RPG по типу Warspear в дальнейшем планируется сделать мультиплеер (если получиться)
пока ведеться разработка буквально самый старт пока нет даже готового кода для тестов

подземалья

# Особенности
Ретро стиль под старые игры (Дань уважения класике)
интерактивность (разнообразное взаимодействия с окружающим миром)
производительность (запуск на любом железе)
Возможные обновления (крост-платформенность)

# Основные моменты для разработки и игры
architecture.md - архитектура всего проекта где какой файл и что к чему превязано ## Просматривать если запутался в файлах
roadmap.md - дорожная карта проекта что должно быть в игре какие задачи стоят ## Просматривать во время прокта
vision.md - текстовая визуализация игры что будет в игре и как это должно выглядить (ланшафт, нпс, логика, диалоги, задумка игры) ## Просматривать что бы держаться плана игры
requirements.txt - файл с требованиями для работы программы (сделанно для виртуальной среды) - ## Просматривать перед запуском программы важные библиотеки
